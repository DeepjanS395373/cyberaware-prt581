\## Project Summary

CyberAware is a web-based social engineering awareness training platform. The system helps learners identify and respond to phishing emails, fake login pages, SMS scams, suspicious links, impersonation attempts, and pretexting. The platform includes training modules, simulation scenarios, quizzes, feedback, progress tracking, and administrator reporting.



\## Development Method

The project follows an Agile-incremental approach. Requirements are written as user stories, prioritised, and organised into sprint-based tasks. The prototype is developed using HTML, CSS, and JavaScript and deployed through GitHub Pages.
