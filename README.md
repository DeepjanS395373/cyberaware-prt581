# CyberAware - PRT581 Software Systems Project

CyberAware is an interactive Social Engineering Awareness Training Platform developed for S126 PRT581 Principles of Software Systems.

The platform is designed to help learners identify phishing emails, fake login pages, suspicious links, SMS scams, impersonation attempts, and other social engineering attacks through interactive training modules, simulations, quizzes, feedback, and progress reporting.
